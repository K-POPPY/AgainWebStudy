1. 블록 레벨 요소와 인라인 레벨 요소

블록 레벨 요소
태그를 사용해 요소를 삽입 했을때 혼자 한줄을 차지하는 것.
한 줄을 차지한 다는 것은 해당 요소의 너비가 100%라는 뜻. 블록 레벨 요소의 왼쪽이나 오른쪽에 다른 요소가 올 수 없음.
📃 블록 요소 MDN

e.g. <h1>, <div>, <p>...
인라인 레벨 요소
태그를 사용해 요소를 삽입 했을때 혼자 한줄을 차지하지 않음.
콘텐츠만큼만 영역을 차지하고 나머지 공간에는 다른 요소가 올 수 있음.
한 줄에 여러개의 인라인 요소를 표시 할 수 있음.

2. 박스 모델의 기본 구성

HTML 블록 레벨 요소(모두 박스 형태) = CSS 박스 모델 요소 의 구성

- margin : 여러 박스 모델 사이의 여백
- border : 박스의 테두리
- padding : 박스와 콘텐츠 영역 사이의 여백
- contensts area: 콘텐츠 영역

3. 콘텐츠 영역의 크기 지정, width height

width, height의 속성값
종류 설명
크기 단위 너비나 높이의 값을 px 나 em 단위로 지정
백분율, % 박스 모델을 포함하는 부모 요소를 기준으로 너빗값이나 높잇값을 백분율(%)로 지정
ㄴ 가변 너비: 부모가body면 웹 브라우저 크기에 따라 달라짐
auto 박스 모델의 너비값과 높잇값이 콘텐츠 양에 따라 자동으로 결정

4. 박스 모델의 크기를 계산, box-sizing

박스 모델이 차지하는 크기: 콘텐츠 영역+패딩+테두리+마진 (width, height 은 마진,패딩,보더를 뺀 콘텐츠 영역의 크기를 지정)
-> 화면에 콘텐츠 배치시 계산이 많아져 복잡해져 border-box 속성 사용.

* border-box: 패딩, 보더까지 포함해 너빗값 지정.
* content-box: 기본값, 콘텐츠 영역만 너비값 지정

5. 박스 모델의 그림자 효과, boxx-shadow

box-shadow: 수평거리(필수값) 수직거리(필수값) 흐림정도(0기본값, 커질수록 부드러움) 번짐정도(양수 크게, 음수 축소) 색상(공백을로 여려개 지정가능) inset(안쪽그림자)

* 수평, 수직값: 양수 오른쪽, 음수 왼쪽에 그림자 만듦

08-2. 테두리 스타일 지정하기

스타일 지정 방향(시계방향): top -> right -> bottom -> left

1. 테두리 스타일, border-style

border-style의 속성값
종류 설명 예시
none 테두리 없, 기본값
hidden 테두리 감춤, 표에서 border-collsapse: collapse 일경우 테두리 안보임
solid 테두리 실선
dotted 테두리 점선
dashed 테두리 짧은 직선
duble 테두리 이중선, 선 사이 간격이 border-width
groove 테두리 홈이 파인 듯 입체 느낌
inset border-collsapse: seperate -> 전체 테이블 창에 박혀있는 모양
border-collsapse: collapse -> grove모양
outset 테두리가 창에서 튀어 나와 있는 모양
ridge 테두리가 창에서 튀어나온 모양

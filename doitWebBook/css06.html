06-3. CSS 기본 선택자 알아보기

1. 태그와 요소의 차이

태그: 말 그대로 태그 자체를 가리키는 말.
요소: 태그를 적용한 것.

<p>텍스트 단락 지정하기</p>

위 소스에서 <p>,</p>태그는 태그 자체를 말하는 것이고 <p>텍스트 단락 지정하기</p> 전체부분을 <p>요소.
위 코드의 적용한 스타일은 p 요소에 적용하는 스타일

2. 선택자

  • 전체 선택자: 문서의 모든 요소애 스타일 지정 | * {스타일 규칙}
  • 타입 선택자: 특정 태그를 사용한 모든 요소에 스타일 적용. 타입 선택자는 태그 이름을 선택자로 사용하므로 태그 선택자, 요소 선택자라고도 함. W3C공식문서는 type selector.
  • id 선택자 : 같은 태그라도 일부는 다른 스타일을 사용하고 싶으면 태그에 id="아이디명"을 지정하면 id를 지정한 요소만 선택 가능. #아이디명 으로 선택자를 선택해 스타일 지정.
    id 선택자는 문서에서 한 번만 적용할수 있음.
    id선택자는 중복해서 적용할수 없으므로 주로 문서의 레이아웃과 관련된 스타일, 웹 요소에 자바스크립트 프로그램을 사용하면서 요소를 구별할때 사용.
  • 클래스 선택자 : id 선택자와 마찬가지로 웹문서의 특정 부분을 선택해 스타일을 지정. class="클래스명"을 태그에 적용 .클래스명 으로 선택해 스타일 지정.
    class선택자는 문서에서 여러 번 적용 가능.
  • 그룹 선택자: 같은 스타일 규칙을 사용하고 싶을때 선택자를 쉼표(,)로 구분해 나열하고 스타일 규칙을 한 번만 정의함. | 선택자, 선택자 {스타일 규칙}
